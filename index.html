계정 및 데이터 삭제 요청

삭제 전 반드시 확인해주세요

삭제 요청이 처리되면 해당 데이터는 복구할 수 없습니다. 요청 접수 후 7일 이내에 처리되며, 처리 완료 시 등록된 이메일로 안내드립니다.

요청자 정보

삭제 요청을 위한 본인 확인 정보를 입력해주세요

삭제 항목 선택

삭제를 원하는 항목을 모두 선택해주세요

계정 삭제 계정 정보(이메일, 이름, 로그인 정보, 설정값) 및 모든 관련 데이터가 영구 삭제됩니다
치료 일지 데이터 삭제 작성된 모든 치료 기록, 세션 노트, 첨부 파일이 삭제됩니다
일정 데이터 삭제 등록된 치료 일정, 반복 일정, 알림 설정이 삭제됩니다
프로필 데이터 삭제 등록된 아동 프로필(이름, 생년월일, 치료 이력) 및 관련 정보가 삭제됩니다
삭제 항목별 상세 데이터
  • 계정 삭제: 이메일, 이름, 로그인 정보, 앱 설정값
  • 치료 일지: 세션 기록, 치료 메모, 첨부 이미지/파일
  • 일정: 치료 일정, 반복 일정, 알림 설정
  • 프로필: 아동 이름, 생년월일, 진단 정보, 치료 이력

추가 요청사항

삭제와 관련한 추가 요청이 있으시면 입력해주세요

위 내용을 확인하였으며, 선택한 데이터가 영구 삭제되어 복구할 수 없음에 동의합니다.
